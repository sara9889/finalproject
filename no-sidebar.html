Proceso de elaboración de la taza

El diseño de nuestra propuesta "Smart Mug" fue diseñada en el programa 3Dmax en el que se crearon 2 cilindros, la base servirá para poder poner todo el sistema requerido para que este pueda alumbrar de acuerdo a la temperatura, se propuso un diseño con formas circulares y rectangulares para darle un estilo futurista, asimismo se creó otro cilindro encima con el fin de que el usuario pueda sacar la taza para su lavado y luego pueda volver a colocarlo mediante presión con la base cuando este lo requiera.
